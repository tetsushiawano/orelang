# syntax
## 四則演算
```
$ 3 + 1 => 4
$ 3 - 1 => 2
$ 3 * 2 => 6
$ 3 / 2 => 1
$ 6 % 3 => 0
```

## 変数定義、変数代入、変数表示
ファイルを作成して、下のプログラムを入力。そして、ファイル名を第一引数に指定して実行。
```
a = 1;
b = 3;
println(a + b);
```
=> 4
